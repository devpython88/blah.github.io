CreateC

An easy-to-use C++ Build-system.

About

CreateC is a easy to use c++ json-base build system

It simplifies stuff like linking Qt5, GTK+ or GTKmm with your app

Normally if you use CMake or Make. You would either have to find the package using the build system or use pkg-config.

CreateC library linking

In CreateC you can simple add a libraries entry in your make.json file.
The entry should be a array with the names of your libraries
For SDL2, raylib, SFML, etc... You just add the name of the library into the entry array
For Qt5, You need to add qt5widgets, qt5core and qt5gui into the libraries entry based on your needs
For GTK+, Simply add gtk3 into the libraries entry
For GTKmm, Simple add gtkmm into the libraries entry

CreateC Example JSON file

Here is a example json file

                {
                    "project": "YourProjectName",
                    "sources": [
                        "src/main.cpp",
                        "src/mymod.h",
                        "src/mymod.cpp",
                        ...
                    ],
                    "libraries": [
                        "SFML",
                        "SDL2",
                        "raylib",
                        "qt5widgets",
                        "qt5core",
                        "qt5gui",
                        "gtk3",
                        "gtkmm"
                    ],
                    "cppstd": "17",
                    "optlvl": 3 // for -O3. It can also be 1 or 2
                }
